# 施餓鬼写真ギャラリー 更新マニュアル

## 📋 概要
施餓鬼法要の写真を毎年追加する手順

---

## 🔄 毎年の更新手順

### ステップ1: 写真の準備（檀家様/住職）

1. **施餓鬼法要の写真を撮影**（8月15日）
2. **フォルダ作成**: `images/shakaki{年度}/`
   - 例: `images/shakaki2026/`
3. **ファイル名を統一規則でリネーム**:
   - フォーマット: `shakaki{年度}_{連番}.jpg`
   - 例: `shakaki2026_01.jpg`, `shakaki2026_02.jpg`, ...
   - 連番は01から始める（ゼロ埋め2桁）
4. **写真をフォルダにアップロード**

---

### ステップ2: コード更新（Antigravityとのやり取り）

**Antigravityへのメッセージ例**:
```
「施餓鬼写真を追加してください」
- 年度: 2026年
- 枚数: 7枚
```

**Antigravityが自動で編集するファイル**:
- `redesign-v2/js/photo-accordion.js`

**追加されるコード例**:
```javascript
const SEGAKI_PHOTO_DATA = {
    2026: { count: 7, prefix: 'shakaki' }, // ← 新規追加
    2025: { count: 5, prefix: 'shakai' },
    2024: { count: 8, prefix: 'shakaki' },
    // ...
};
```

---

### ステップ3: 動作確認

1. **ブラウザをリフレッシュ**（Ctrl + F5）
2. **施餓鬼モーダルを開く**
   - トップページ → 「お盆・施餓鬼会法要」→「詳しく見る」
3. **新しい年度の動画下に「📷 写真を見る」ボタンがあるか確認**
4. **ボタンをクリックして写真が表示されるか確認**

## ⚠️ 注意事項

### ファイル名の規則
- **必ず統一**: `{prefix}{年度}_{連番}.jpg`
- **連番はゼロ埋め**: 01, 02, 03, ... (1桁の場合は01にする)
- **prefix**:
  - 基本: `shakaki`
  - 2025年のみ例外: `shakai`（既存ファイル名に合わせる）

### 画像形式
- **推奨**: JPG形式
- **サイズ**: 横幅1200px以上推奨（自動でリサイズされます）

### HTMLの動画部分
- 動画はそのまま手動でHTMLに追加（WIX/Vimeo連携）
- 写真のみ自動読み込み
